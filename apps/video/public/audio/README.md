# Audio Assets for ShortTeaserB

Add the following audio files to this folder:

## Required Files

| File | Duration | Description | Timing |
|------|----------|-------------|--------|
| `drone.mp3` | ~9s | Deep sub-bass drone/tension builder | Plays throughout |
| `glitch-clicks.mp3` | ~5s | Digital clicks/beeps for number flashing | 0:00 - 5.3s |
| `impact.mp3` | ~1s | Bass drop/hit when stopping on "??" | At 4.7s |
| `whoosh.mp3` | ~1s | Swoosh sound for text reveal | At 5.3s |
| `bass-hit.mp3` | ~1s | Deep 808/bass hit for "SOON" | At 7.0s |

## Tips

1. Keep files short - trim to only what's needed
2. Use MP3 or WAV format
3. Normalize audio levels before adding
4. The drone should fade out naturally or be trimmed to ~9s
